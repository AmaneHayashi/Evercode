# FPGA（基于FPGA的点阵游戏）

> 大三上学期《数字电路与逻辑设计实验（下）》代码内容

## 程序说明
 - *core*-核心程序（从这里打开整个工程）
 - *dvdr*-分频器
 - *pic*-图案显示（*pic_rdm*创建随机数，*pic_st*随机状态，*pic_p*图案）
 - *dzn*-点阵（*dzn_st*点阵状态，*dzn_lg*点阵逻辑）
 - *key*-键盘（*key_st*键盘状态，*key_lg*键盘逻辑）
 - *num*-数码管
 - *xns*-庆祝（*xns_add*庆祝计数器，*xns_pic*庆祝图案）
 - *extr*-附加（*extr_st*音乐状态，*extr_beep*音乐）
